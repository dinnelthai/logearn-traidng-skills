# 核心功能流程图

## 1️⃣ Fibonacci自动交易流程

### 整体流程
```
┌─────────────────────────────────────────────────────────────┐
│                   Fibonacci自动交易                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 启动 run_fibonacci_trade(ca, total_capital)             │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 创建 SingleTradeBot                                      │
│     - 初始化 TradeExecutor                                   │
│     - 初始化 PositionManager                                 │
│     - 初始化 ProfitManager                                   │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 首次获取全量K线（自动翻页）                               │
│     - KlineService.get_all_klines()                          │
│     - 存入 KlineCache                                        │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  4. 主循环（每60秒）                                         │
└─────────────────────────────────────────────────────────────┘
        ↓                   ↓                   ↓
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 增量更新K线   │  │ 检测买入信号  │  │ 检测卖出信号  │
└──────────────┘  └──────────────┘  └──────────────┘
        ↓                   ↓                   ↓
┌──────────────────────────────────────────────────────────┐
│ KlineCache.get_klines() - 只获取新增K线                   │
└──────────────────────────────────────────────────────────┘
                            ↓
┌──────────────────────────────────────────────────────────┐
│ fib_signal() - 检测买入/卖出/止损信号                      │
└──────────────────────────────────────────────────────────┘
```

### 买入流程详解
```
┌─────────────────────────────────────────────────────────────┐
│                      买入信号检测                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 检测波峰波谷                                             │
│     - zigzag_pivots() 找到最近的波峰和波谷                   │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 计算Fibonacci档位                                        │
│     - fib_entry_levels(swing_high, swing_low)                │
│     - 返回: buy_618, buy_786, buy_861, stop                  │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 检查价格穿透                                             │
│     - 当前价 <= buy_618 * 1.005 (0.5%容差)                   │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  穿透？        │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 继续监控
        │ 触发买入信号    │
        └────────────────┘
                ↓
┌─────────────────────────────────────────────────────────────┐
│  4. 执行买入                                                 │
│     - check_can_buy() 检查是否可以买入                       │
│     - calculate_position_size() 计算买入金额                 │
│     - executor.buy() 执行买入                                │
└─────────────────────────────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────────────────────────┐
│  5. 更新状态                                                 │
│     - 记录 entry_price, entry_amount                         │
│     - 标记 tiers_bought                                      │
│     - 锁定 swing_high, swing_low                             │
└─────────────────────────────────────────────────────────────┘
```

### 卖出流程详解
```
┌─────────────────────────────────────────────────────────────┐
│                      卖出信号检测                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 计算AO值                                                 │
│     - calc_ao(klines) 计算Awesome Oscillator                 │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 检测AO卖出信号                                           │
│     - ao_sell_signal(ao_values, entry_price, current_price)  │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │ AO绿转红？     │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 检查FIB卖出
        │ AO >= 35k?     │
        └────────┬───────┘
            Yes ↓   ↓ No
    ┌──────────┐  ┌──────────────────┐
    │ 立即卖出  │  │ 收益率 > 50%?     │
    └──────────┘  └────────┬─────────┘
                      Yes ↓   ↓ No
                  ┌──────────┐ └→ 不卖出
                  │ 卖出      │
                  └──────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 执行卖出                                                 │
│     - executor.sell(ca, percentage)                          │
│     - 卖出比例根据AO值决定（30%-50%）                        │
└─────────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│  4. 检查是否全部卖出                                         │
│     - 如果全部卖出 → 交易完成，退出                          │
│     - 如果部分卖出 → 继续监控                                │
└─────────────────────────────────────────────────────────────┘
```

### 止损流程
```
┌─────────────────────────────────────────────────────────────┐
│                      止损检测                                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 计算止损价                                               │
│     - stop_price = swing_high - (swing_high - swing_low) * 0.92 │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 检查当前价                                               │
│     - current_price <= stop_price?                           │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  触发止损？    │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 继续监控
        │ 全部卖出        │
        └────────────────┘
                ↓
        ┌────────────────┐
        │ 交易结束        │
        └────────────────┘
```

## 2️⃣ RSI定投机器人流程

### 整体流程
```
┌─────────────────────────────────────────────────────────────┐
│                   RSI定投机器人                              │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 启动 run_rsi_dca(ca, total_capital, buy_amount)         │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 创建 RSIDCABot                                           │
│     - 初始化 TradeExecutor                                   │
│     - 初始化 PositionManager                                 │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 主循环（每5分钟）                                        │
└─────────────────────────────────────────────────────────────┘
        ↓                   ↓                   ↓
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 获取1小时K线  │  │ 计算RSI       │  │ 执行交易      │
└──────────────┘  └──────────────┘  └──────────────┘
```

### RSI买入流程
```
┌─────────────────────────────────────────────────────────────┐
│                      RSI买入检测                             │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 获取1小时K线（最近100根）                                │
│     - get_klines(ca, interval='1h', page_size=100)           │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 计算RSI                                                  │
│     - calculate_rsi_series(closes, period=14)                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 检查RSI值                                                │
│     - RSI < 30 (超卖)                                        │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  RSI < 30?    │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 继续监控
        │ 检查资金        │
        └────────┬───────┘
                 ↓
┌─────────────────────────────────────────────────────────────┐
│  4. 检查是否可以买入                                         │
│     - 已投入资金 < total_capital                             │
│     - 持仓价值 < max_position_ratio * total_capital          │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  可以买入？    │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 等待下次
        │ 执行买入        │
        │ buy_amount SOL │
        └────────────────┘
                ↓
┌─────────────────────────────────────────────────────────────┐
│  5. 更新状态                                                 │
│     - 记录买入价格和数量                                     │
│     - 累计已投入资金                                         │
└─────────────────────────────────────────────────────────────┘
```

### RSI卖出流程
```
┌─────────────────────────────────────────────────────────────┐
│                      RSI卖出检测                             │

                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 计算RSI                                                  │
│     - RSI > 70 (超买)                                        │
└─────────────────────────────────────────────────────────────┘
                            ↓

                    │  RSI > 70?    │
                    └───────┬───────┘

        ┌────────────────┐   └→ 继续监控

        └────────┬───────┘
                 ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 检查是否有持仓                                           │
│     - get_positions() 获取当前持仓                           │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  有持仓？      │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 继续监控
        │ 全部卖出        │
        │ percentage=1.0 │
        └────────────────┘
                ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 重置状态                                                 │
│     - 清空买入记录                                           │
│     - 重置已投入资金                                         │

```

```
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 启动 run_ao_monitor(ca, entry_price, sell_percentage)   │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 验证持仓                                                 │
│     - executor.get_positions()                               │
│     - 检查是否持有该代币                                     │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  有持仓？      │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 退出（无持仓）
        │ 开始监控        │
        └────────────────┘
                ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 主循环（每60秒）                                         │
└─────────────────────────────────────────────────────────────┘
        ↓                   ↓                   ↓
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 获取5分钟K线  │  │ 计算AO        │  │ 检测卖出信号  │
└──────────────┘  └──────────────┘  └──────────────┘
```

### AO卖出信号检测
```
┌─────────────────────────────────────────────────────────────┐
│                      AO卖出信号检测                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 获取5分钟K线（最近200根）                                │
│     - get_klines(ca, interval='5m', page_size=200)           │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 计算AO值                                                 │
│     - calc_ao(klines)                                        │
│     - AO = SMA(5) - SMA(34) of median price                  │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 检测绿转红                                               │
│     - color_n1 = "green" if ao_n1 >= ao_n2                   │
│     - color_0 = "red" if ao_0 < ao_n1                        │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │ 绿转红 且      │
                    │ ao_0 > 0?     │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   └→ 继续监控
        │ 检查AO阈值      │
        └────────┬───────┘
                 ↓
        ┌────────────────────────┐
        │  AO >= 35k?            │
        └────────┬───────────────┘
            Yes ↓   ↓ No
    ┌──────────┐  ┌──────────────────┐
    │ 立即卖出  │  │ 有买入价？        │
    └──────────┘  └────────┬─────────┘
                      Yes ↓   ↓ No
              ┌──────────────┐ └→ 不卖出
              │ 收益率 > 50%? │   （保守模式）
              └────────┬─────┘
                  Yes ↓   ↓ No
              ┌──────────┐ └→ 不卖出
              │ 卖出      │
              └──────────┘
```

### 卖出执行流程
```
┌─────────────────────────────────────────────────────────────┐
│                      执行卖出                                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 输出信号信息                                             │

└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐

                    ┌───────┴───────┐
                    │  卖出成功？    │
                    └───────┬───────┘

        ┌────────────────┐   ┌────────────────┐
        │ 停止监控        │   │ 继续监控        │

        └────────────────┘   └────────────────┘
```

### 多币监控流程
```
┌─────────────────────────────────────────────────────────────┐
│          run_ao_monitor_multi(configs)                       │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  1. 验证所有代币的持仓                                       │
│     - 过滤掉没有持仓的代币                                   │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 主循环（轮询所有代币）                                   │
└─────────────────────────────────────────────────────────────┘
        ↓
┌─────────────────────────────────────────────────────────────┐
│  for each config in configs:                                 │
│      if config.ca in sold_cas:                               │
│          continue  # 已卖出，跳过                            │
│      check_ao_signal(config)  # 检查AO信号                   │
│      if signal:                                              │
│          sell(config)  # 执行卖出                            │
│          sold_cas.add(config.ca)  # 标记已卖出               │
└─────────────────────────────────────────────────────────────┘
        ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 检查是否全部卖出                                         │
│     - if len(sold_cas) == len(configs): 退出                 │
└─────────────────────────────────────────────────────────────┘
        ↓
┌─────────────────────────────────────────────────────────────┐
│  4. 等待下次检查                                             │
│     - time.sleep(check_interval)                             │
└─────────────────────────────────────────────────────────────┘
```

## 🔧 核心模块交互流程

### K线获取流程
```
┌─────────────────────────────────────────────────────────────┐
│                      K线获取流程                             │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  调用方                                                      │
│  get_klines(ca, interval, page_size)                         │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  KlineService                                                │
│  - 单例模式，全局共享                                        │
└─────────────────────────────────────────────────────────────┘
                            ↓
                    ┌───────┴───────┐
                    │  有缓存？      │
                    └───────┬───────┘
                    Yes ↓       ↓ No
        ┌────────────────┐   ┌────────────────┐
        │ 返回缓存数据    │   │ 调用API获取     │
        └────────────────┘   └────────┬───────┘
                                      ↓
                            ┌─────────────────┐
                            │ KlineFetcher    │
                            │ 调用GMGN API    │
                            └────────┬────────┘
                                     ↓
                            ┌─────────────────┐
                            │ 存入缓存         │
                            │ 返回数据         │
                            └─────────────────┘
```

### 交易执行流程
```
┌─────────────────────────────────────────────────────────────┐
│                      交易执行流程                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  调用方                                                      │
│  executor.buy(ca, amount_sol) / executor.sell(ca, percentage)│
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  TradeExecutor                                               │

                            ↓
┌─────────────────────────────────────────────────────────────┐
│  _logearn_swap()                                             │
│  - 调用LogEarn CLI                                           │
│  - 参数：token_in, token_out, amount_in                      │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  LogEarn CLI                                                 │
│  - 执行链上交易                                              │
│  - 返回交易结果                                              │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  返回 TradeResult                                            │
│  - success: bool                                             │
│  - code: int                                                 │
│  - message: str                                              │
└─────────────────────────────────────────────────────────────┘
```
